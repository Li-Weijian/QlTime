# QlTime
### 项目说明
此项目是前后端分离结构的后台。
前端代码参考https://github.com/Li-Weijian/lovezz-web
目前开发的功能有： 定时发送短信、纪念日、情侣待办、图库、登录、朋友圈功能，。

## 技术栈 
Springboot 2.1.6.RELEASE + Spring + Mybatis-plus 2.1.8 + 
Mysql 5.7 + 阿里云OSS对象存储 + 腾讯短信


## 运行说明
问： 如何运行
答： 配置好Mysql，运行即可。

## 打包
mvn clean package -Dmaven.test.skip=true
